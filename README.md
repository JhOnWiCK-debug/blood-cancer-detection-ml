# Blood Cancer Detection Using Machine Learning

## Overview

This project develops a machine learning-based blood cancer detection system using healthcare datasets and comparative analysis of multiple classification algorithms.

## Algorithms Evaluated

- Support Vector Machine (SVM)
- Decision Tree (DT)
- K-Nearest Neighbors (KNN)
- Logistic Regression (LR)
- Gradient Boosting (GB)

## Results

| Model | Accuracy |
|---------|---------|
| Gradient Boosting | 99.84% |
| Logistic Regression | 96.12% |
| Decision Tree | 96.12% |
| KNN | 95.31% |
| SVM | 85.18% |

## Statistical Validation

- Independent Samples T-Test
- SPSS Analysis
- p-value = 0.002
- Performance evaluated across multiple experimental runs
